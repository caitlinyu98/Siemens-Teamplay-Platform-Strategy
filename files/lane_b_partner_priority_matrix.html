Lane B partner shortlist

Aidoc (aiOS)
Critical findings triage · CT + MR · enterprise scale
Priority now
FDA Breakthrough Device 2025 200k+ cases proven at WellSpan Epic + PACS integrated Multi-pathology single submission Requires workflow integration depth
Strongest enterprise-proven triage platform. Works across CT and MR — maps directly onto Siemens scanner fleet. The FDA Breakthrough Designation means regulatory credibility accelerates. Ties fleet scanner utilization data to clinical outcome data — the ROI story Fleet customers need.
Fleet hook: scanner-level alert volume + triage time reduction = measurable ROI in Fleet dashboard
Nuance PowerScribe (Microsoft)
Radiology reporting + AI impression drafting · de facto standard
Priority now
De facto reporting standard Generative AI drafting in production Already in most target hospitals Microsoft — potential strategic conflict Platform integration, not listing
Not a marketplace app — it's the connective tissue. Any AI app surfaced through Teamplay that feeds into PowerScribe gains instant workflow legitimacy. Siemens should treat this as a platform integration, not a vendor listing. Goal: Teamplay-deployed apps become first-class citizens in the PowerScribe worklist.
Fleet hook: reporting turnaround time per scanner modality becomes a trackable Fleet KPI
Subtle Medical (SubtleMR / SubtlePET)
AI image enhancement · scan time reduction · MR + PET
Add in Q2
Scanner-native value prop Throughput + image quality gains No clinical overlap with Siemens Easy ROI: scan time is measurable Narrow modality scope (MR, PET)
Rare category where ROI proof is already in the Fleet data — scanner utilization, exam throughput, protocol time. SubtleMR increases MR scanner output without adding hardware. Easiest co-marketing story: "get more scans from your existing MAGNETOM fleet."
Fleet hook: before/after throughput per MAGNETOM unit — Fleet already has this data
Cerebriu
Brain MRI triage + protocol automation · MR-native
Add in Q2
MR-native — MAGNETOM fit Triage + protocol automation CARPL.ai ecosystem integrated Smaller scale than Aidoc Narrower scope (brain MRI only)
Strong complement to Aidoc for the MR fleet. Protocol automation reduces technologist burden and scan variability across fleet sites — directly relevant to multi-site enterprise customers. CARPL.ai integration means it already has a standardized API surface.
Fleet hook: protocol compliance rate per MAGNETOM unit across enterprise sites
Blackford Analysis
AI orchestration layer · 140+ apps · multi-vendor PACS
Strategic watch
Proven enterprise AI orchestration Partners with Philips, Mach7, Rhapsody Competes with Teamplay's platform ambition If they win, Teamplay becomes redundant
Blackford is building exactly what Teamplay is supposed to become. Partner only if Siemens accepts Blackford as the orchestration layer and Teamplay as the scanner/Fleet data feed. If Siemens wants to own orchestration itself, Blackford is a competitor, not a partner.
Decision: does Siemens want to own the app layer — or just feed data into someone else's?
— avoid —
Intelerad
PACS + cloud imaging — acquired by GE HealthCare Nov 2025 for $2.3B · close expected H1 2026
Avoid
Any integration work Siemens does with Intelerad now builds distribution for GE's platform. The window to use Intelerad as a neutral partner has closed. Do not invest in this relationship.
The real prerequisite before any of this matters
The hospital already trusts Fleet. Fleet is where they live. The marketplace has no trust and no traffic. The path to adoption: Fleet dashboard surfaces relevant app recommendations per scanner modality → one-click trial → standardized deployment through Fleet's existing service path → ROI visible in Fleet's existing analytics. No standalone marketplace adoption campaign will work until Fleet is the entry point.
